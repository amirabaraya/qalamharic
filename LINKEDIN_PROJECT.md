## Description

FidelAmharic is a full-stack Amharic learning platform I built for native English speakers who are starting Amharic from the beginning. The product includes learner accounts, database-backed progress, gamified lessons, XP, streaks, hearts, badges, course maps, pronunciation practice, flashcards, spaced review, and an admin studio for managing courses, lessons, users, plans, and access.

I designed the interface to feel modern, warm, and premium while drawing inspiration from Ethiopian identity, Ge'ez script beauty, earthy green, gold, cream, and deep charcoal. The goal was to create a serious learning product that feels culturally distinctive and easy for beginners to use.

Built with Next.js, TypeScript, React, Tailwind CSS, Prisma, PostgreSQL, NextAuth, Stripe-ready subscriptions, and Vercel deployment.
